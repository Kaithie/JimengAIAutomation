# 视频片段拆分工具

一款基于AI的视频片段拆分桌面工具，帮助视频创作者将故事描述或旁白拆分成适合AI视频生成的分镜提示词。

## 功能特点

- **智能拆分**：输入视频描述或旁白，AI自动拆分成5-15秒的视频片段提示词
- **素材管理**：支持上传人物图片（最多5个）、场景图片、声线素材
- **素材引用**：AI生成的提示词自动引用上传的素材，格式为@[文件名.扩展名]
- **表格编辑**：片段以表格形式展示，支持修改提示词、上传视频、生成九宫格
- **即梦视频生成**：点击"生成视频"按钮，自动调用即梦AI生成视频
- **九宫格生成**：从上传的视频中提取9帧，拼接成九宫格并添加剧情简述
- **多平台支持**：支持通义千问、DeepSeek、OpenAI、豆包等多个AI平台

## 系统要求

- Windows 10/11（64位）
- 无需安装Python环境（使用打包版）

## 安装使用

### 方式一：直接使用打包版（推荐）

1. 下载 `视频片段拆分工具.exe`
2. 双击运行即可

### 方式二：从源码运行

1. 安装Python 3.9+
2. 克隆或下载项目
3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
4. 运行：
   ```bash
   python main.py
   ```

### 方式三：自行打包

运行打包脚本：
```bash
build.bat
```

打包后的可执行文件位于 `dist` 目录。

## 使用指南

### 1. 配置API

首次使用需要配置AI API：

1. 点击「设置」按钮
2. 选择AI平台（推荐通义千问或DeepSeek）
3. 输入API Key
4. 点击「测试连接」确认配置正确
5. 保存设置

### 2. 准备素材

在左侧素材面板上传素材：

- **人物图片**：最多5个，上传时填写名称和描述
- **场景图片**：无数量限制
- **声线素材**：音频文件

### 3. 输入描述

在主界面文本框中输入视频描述或旁白文本。

### 4. 生成片段

点击「拆分片段」按钮，AI将自动：
- 分析描述内容
- 拆分成多个5-15秒的片段
- 为每个片段生成视频生成提示词
- 自动引用已上传的素材

### 5. 管理片段

在片段表格中可以：
- **编辑提示词**：直接在文本框中修改
- **上传视频**：为片段上传已生成的视频
- **生成视频**：点击"生成视频"按钮，自动调用即梦AI生成视频
- **生成九宫格**：从上传的视频生成九宫格截图
- **删除片段**：删除不需要的片段
- **新增片段**：在指定位置插入新片段

### 6. 即梦视频生成

点击片段的"生成视频"按钮，工具将自动：

1. 打开即梦AI视频生成网站
2. 切换到"全能参考"模式
3. 按顺序上传提示词中引用的素材文件
4. 填写提示词（自动处理 @[文件名] 格式的引用）
5. 提交视频生成请求

**注意事项**：
- 首次使用需要在浏览器中登录即梦账号
- 登录状态会自动保存，下次无需重新登录
- 视频生成需要时间，请在浏览器中查看进度
- 确保提示词中引用的文件已存在于项目的 assets 目录中

## 提示词格式

工具生成的提示词遵循以下格式：

```
镜头：【景别+运镜】 环境：@[场景.png] [场景描述] 角色分动：@[人物.png] [外观] 正在 [动作] 细节：[微表情与情绪] 光影：[光源+质感+色调] 音效：[环境音+声响] 台词：["对白"]
```

## 注意事项

1. API Key会保存在本地配置文件中，请妥善保管
2. 九宫格生成需要安装moviepy库
3. 即梦视频生成需要安装playwright库并运行 `playwright install chromium`
4. 视频片段时长建议5-15秒，AI会根据旁白长度自动调整
5. 人物素材最多支持5个，每个文件只能引用一次
6. 即梦账号登录状态保存在 `~/.jimeng-browser-data/` 目录
